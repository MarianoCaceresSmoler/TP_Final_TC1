    \begin{abstract}
        En este informe se desarrolla el Trabajo Práctico Final de Teoría de Circuitos I. Se presenta una interfaz gráfica en Python para procesar archivos CSV, el análisis transitorio de un circuito RLC, el estudio de filtros de segundo orden y el diseño de un filtro pasabanda para un primer filtrado de la voz humana. El proyecto integra análisis teórico, simulación en LTspice, mediciones de laboratorio y diseño de PCB en Altium.
    \end{abstract}
